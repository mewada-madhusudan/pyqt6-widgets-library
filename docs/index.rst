PyQt6 Widgets Library Documentation
===================================

Welcome to the PyQt6 Widgets Library documentation. This library provides a comprehensive collection of **polished, reusable PyQt6 widgets** for building modern desktop applications with consistent UI/UX.

Features
--------

* **50+ Pre-built Widgets** across 7 categories
* **UX-First Design** with modern styling and animations
* **Modular Architecture** for easy customization and extension
* **Theme Management** with light/dark mode support
* **Comprehensive Documentation** with usage examples
* **Cross-Platform** support (Windows, macOS, Linux)

Widget Categories
-----------------

:doc:`api/cards`
    Information display cards, profile cards, statistics cards, and more

:doc:`api/navigation`
    Sidebar navigation, breadcrumbs, tabs, command palette, and pagination

:doc:`api/feedback`
    Notifications, status indicators, progress overlays, and tooltips

:doc:`api/data`
    Data tables, timelines, kanban boards, charts, and file explorers

:doc:`api/user`
    User avatars, chat bubbles, ratings, and social components

:doc:`api/forms`
    Search boxes, inline editors, tag inputs, and form wizards

:doc:`api/utility`
    Floating action buttons, settings panels, and utility widgets
